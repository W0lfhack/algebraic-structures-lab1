## Описание
Реализация задач уровней **Easy** и **Normal** из лабораторной работы по алгебраическим структурам:
- Теория чисел (палиндромы, простые, функция Эйлера)
- Теория групп (симметрические группы, циклические подгруппы, изоморфизмы)
- Полиномы над конечными полями (корни, приводимость, НОД, обратные элементы)

Используемые инструменты: **Python**, **SageMath**, **SymPy**.

## Структура
- `easy_level.py` — задачи уровня Easy
- `normal_groups.py` — задачи уровня Normal раздел "Группы"
- `normal_polynoms.py` - задачи уровня Normal раздел "Полиномы"

## Запуск
Для Easy уровня достаточно:
```bash
python easy.py
```
Для запуска задач уровня Normal требуется **SageMath**:
```bash
sage --python normal_groups.py
sage --python normal_polynoms.py
